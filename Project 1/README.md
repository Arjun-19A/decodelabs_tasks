# 🚀 DecodeLabs Project 1 - Responsive Architecture

## 📌 Project Name

**TaskFlow AI - Responsive SaaS Landing Page**

A modern, responsive landing page built using **HTML5, CSS3, and Vanilla JavaScript** as part of the DecodeLabs Full Stack Development Internship.

---

## 📖 Overview

TaskFlow AI is a fictional AI-powered project management platform designed to demonstrate responsive web design, semantic HTML, modern CSS layouts, and JavaScript interactivity.

This project follows a **mobile-first approach** and implements responsive design principles without using any frontend frameworks.

---

## ✨ Features

- 📱 Fully Responsive Design
- 🌙 Dark / Light Mode
- 🍔 Mobile Navigation Menu
- ✨ Smooth Scroll Animations
- 📊 Animated Statistics
- 💳 Pricing Cards
- 💬 Testimonials Section
- 📞 Contact Form
- 🔝 Back to Top Button
- 🎨 Modern UI Design
- ♿ Semantic HTML & Accessibility
